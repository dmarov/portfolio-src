Career

As a start of my career I've got solid math and computer science background in NARFU "Applied Mathematics and Computer Science" which I've graduated in 2017. At that time participated in math and programming olympiads. As a base programming language I regard C++ but also I learned C, Java, Python, Assembly.

Short after that I've started my career as backend developer. It was my first experience of commercial software development. That's when I got acquainted with PHP and MySQL, as well as frontend technologies such as Gulp, Webpack, Sass, Babel.

Then the time has come when native Javascript was not enough in order to keep implementing complex solutions in efficient way. That's when I made decision to learn Vue framework which payed out later.

Path lead to resolve more complex problems related to outdated, unmaintainable infrastructure. Being experienced with Node.js I've implemented microservices. There's some of them:

  • SMS sending gateway - rest API to send SMS notifications using GSM gateway
  • PDF generator - REST API to convert HTML page to PDF (most useful for printing invoices)
  • JWT token generator - REST API to generate JWT token pairs (useful for authentication)

These services was very useful for each new website.

Since each service required separate environnment it required a lot of administering efforts, which I was also responsible for. I already had a lot knowledge about FreeBSD operating system, so it was time to apply this knowledge in practice.

Then I decided to specialize in frontend and move to big projects. That's when I've learned Typescript and Angular which became part of my main technology stack till current day. Such aspects as Site Engine Optimization (SEO), implementing UI test, pixel perfect markup, cross browser development, accessibility - this is what I'm most focused on at work.
