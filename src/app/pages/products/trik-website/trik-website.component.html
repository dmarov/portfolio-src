Trik Website

Trik is a pharmacy network from Krasnodar, Russian Federation. The website is an online shop + CMS.

Visit

Trik needed web store that would serve their 5 pharmacies. It required to create custom logic for catalog, cart and orders. Their site has complicated search by dynamically assignable properties tags, and any logical expression which can be constructed from any of properties or tags. Search result can be filtered by dynamically assignable properties.

I developed backend and frontend.

Technologies I used:

  • Vue
  • Vuex
  • Webpack
  • PHP
  • MySQL
  • Gulp
